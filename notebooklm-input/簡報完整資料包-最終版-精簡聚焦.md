### Part 2：6 個真實痛點（7 分鐘）⭐ 核心

#### **場景 1：做提案當機（1 分鐘）**

**痛點**：
```
做提案 2 小時
電腦當機
沒存檔
全部重來
```

**我們的解法**：
```
系統 3 分鐘生成
雲端自動存檔
```

**意外收穫**：省了 2 小時

---

#### **場景 2：同一份名單輸入 5 次（1 分鐘）**

**痛點**：
```
飯店版（英文名字、房型）
內部版（成本、利潤）
領隊版（景點、集合時間）
航空版（護照號碼）
保險版（身分證字號）

每一份都要重新複製貼上
```

**我們的解法**：
```
輸入 1 次
系統自動生成 5 種版本
```

**意外收穫**：不用重複勞動了

---

#### **場景 3：找不到正確版本（1 分鐘）**

**痛點**：
```
資料夾裡 10 個檔案：
日本行程_v1.docx
日本行程_v2.docx
日本行程_最終版.docx
日本行程_最終版(2).docx
...

客戶改需求，不知道哪個是最新的
給領隊錯誤版本，行程大亂
```

**我們的解法**：
```
訂單、行程、報價綁在一起
永遠只有一個最新版
改了會自動更新
```

**意外收穫**：不會版本混亂了

---

#### **場景 4：老闆出國簽不了合約（1 分鐘）**

**痛點**：
```
老闆帶團在日本
客戶突然說要簽約
老闆：「我不在台灣，合約沒辦法蓋章」
客戶：「那我找別家」
訂單飛了
```

**我們的解法**：
```
電子簽名
手機就能簽
老闆在國外也能處理
```

**意外收穫**：不會錯過訂單了

---

#### **場景 5：客戶打不開檔案（1 分鐘）**

**痛點**：
```
發 PDF 給客戶
客戶：「我手機打不開」
重新傳 Word
客戶：「版面跑掉了」
截圖傳給他
客戶：「字太小看不清楚」
半夜 11 點，印紙本拍照傳給他
```

**我們的解法**：
```
客戶自己登入系統
線上查看，不用下載
任何裝置都能看
```

**意外收穫**：不用半夜起床了

---

#### **場景 6：忘了客人生日（1 分鐘）**

**痛點**：
```
帶團去日本
忘了客人今天生日
沒準備蛋糕、沒唱生日歌
客人不開心，回來打 1 顆星
「其他旅行社都記得我生日，只有你們忘了」
```

**我們的解法**：
```
系統提前 3 天提醒
自動檢查：
- 客人生日
- 護照快過期
- 簽證到期
- 特殊需求（素食、輪椅）
```

**意外收穫**：不會遺漏了

> 
> B 方案第一年平均每月才 250 元。

### **Q2：「戴爾斯是做什麼的？」**

**A**：
> 戴爾斯有限公司是我們的合作夥伴，

> 
> 所以這套系統不只有功能，還有質感。

### **Q3：「我們人少，適合嗎？」**

**A**：
> 人少更適合！
> 
> B 方案適合 6-10 人，
> 
> 配合補助，平均每月 250 元。
> 
> 而且系統可以省很多人工，

### **Q4：「系統會不會很難用？」**

**A**：

### **Q5：「我們已經有 ERP 了」**

**A**：
