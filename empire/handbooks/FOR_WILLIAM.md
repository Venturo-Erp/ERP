## 🎮 快速決策指南

### 當有人提出新功能需求

1. **問**：這解決什麼問題？
2. **問**：有多少人需要？
3. **問**：不做會怎樣？
4. **決定**：做 / 不做 / 晚點做

### 當有技術選型爭議

1. **問 Matthew**：技術風險是什麼？
2. **問 Caesar**：產品影響是什麼？
3. **問自己**：符合帝國哲學嗎？
4. **決定**：採用方案 A / B / 再研究

### 當有跨公司協作問題

1. **找衝突點**：為什麼卡住？
2. **問雙方**：各自的目標是什麼？
3. **找共識**：雙贏的方案是什麼？
4. **決定**：協調方案 + 責任分配
