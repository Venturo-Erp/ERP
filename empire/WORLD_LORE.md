## 📜 創世神話

太初，**威廉（William）**——創世神，俯瞰混沌大地。

祂看見旅人們在混亂中迷失：

- 情報散落在無數羊皮紙上（Excel、紙本、Line 訊息）
- 冒險者不知道任務進度（飯店訂了沒？車訂了嗎？）
- 附屬國收到的委託殘缺不全（傳真模糊、電話漏接）
- 金幣流向無人追蹤（誰付了？誰還沒付？）

於是，創世神說：

> **「讓所有真相匯聚在一棵樹上。
> 所有人看的是同一棵樹，只是站在不同方向。」**

祂種下了**世界樹（Core Table）**——`tour_itinerary_items`。

世界樹是一切的根源，所有的情報、金幣、任務狀態都刻在樹上。
不管你是冒險者、附屬國的工匠、還是遠方的商人，
你看到的都是**同一棵樹的不同面**。

這就是 **Venturo 帝國**的誕生。

## 🎯 帝國的野心

> **統一旅遊業的制度、表格、藍圖。**

不只是做一套 ERP，是要建立整個旅遊產業的**標準作業系統**。
所有旅行社、所有 Local、所有供應商，用同一套世界法則運作。

## 🌳 世界樹（Core Table）

世界樹是帝國的中心，所有真相的根源。

```
🌳 世界樹 (tour_itinerary_items)
│
├── 🍃 東面（行程表視角）
│   看到：天數、路線、景點、住宿安排
│   操作：可以修剪枝葉（編輯行程）
│
├── 🍃 南面（報價單視角）
│   看到：每個項目的成本
│   操作：在樹幹上刻下價格（填成本）
│
├── 🍃 西面（需求單視角）
│   看到：哪些需要委託、委託給誰
│   操作：摘下果實送出去（發需求單）
│
├── 🍃 北面（確認單視角）
│   看到：已確認的項目彙整
│   操作：只讀（交給領隊的交接書）
│
└── 🍂 落葉（結帳單視角）
    看到：估價 vs 實際差異
    操作：記錄最終結果（賺了還是賠了）
```

**世界法則**：

> 「不論你從哪個方向看世界樹，看到的都是同一棵樹。
> 在東面修剪的枝葉，從西面也看得到變化。
> 這就是世界的真相只有一個。」

## ⚔️ 任務系統

### 主線任務：一個團的完整生命週期

```
【第一章：冒險的開始】
  勇者（業務）在公會佈告欄貼出新任務
  → 建立旅遊團（開團）

【第二章：路線規劃】
  勇者在世界樹上刻下路線
  → 行程表編輯（核心表寫入）

【第三章：報酬估算】
  勇者估算每段路的花費
  → 報價單填成本（$5,000 估價）
  客人同意 → 任務正式啟動

【第四章：委託發布】⭐ 現在在做的
  盜賊（助理）從世界樹摘下果實
  → 勾選項目 → 指定委託對象 → 發出委託書

  兩種委託模式：
  A) 直接委託：每個工匠各一張（飯店、餐廳、車行）
  B) 打包委託：勾選多項 → 整包送給附屬國

【第五章：等待回音】
  工匠/附屬國收到委託 → 回覆確認 + 價格
  → 盜賊回填到世界樹
  → 業務估價 $5,000 → 實際 $4,983（直接覆蓋）
  → 世界樹提示：「多賺 $17 💰」

【第六章：交接與出發】
  所有委託確認完畢
  → 世界樹的北面自動長出「確認單」
  → 交給領隊 → 出發！

【第七章：歸來與結算】
  領隊回來 → 記錄實際費用
  → 世界樹落葉（結帳單）
  → 這次冒險賺了還是賠了？
```

## 🏘️ 附屬國系統

### 附屬國的誕生

當冒險公會發出打包委託，附屬國收到的那一刻：

- 世界樹**分出一根枝條**（fork）到附屬國的土地上
- 這根枝條跟母樹同結構，但長在不同的土地上（workspace_id）
- 附屬國可以修剪枝條（微調行程：餐廳公休換順序）
- 母樹看得到枝條的變化（diff 追蹤）

### 附屬國任務板

```
📩 委託收件匣
  「角落旅行社 發來新委託！」
  「HAN260925A — 越南5天4夜 — 20人」
  「需求：交通✅ 餐廳✅ 活動✅ 住宿❌（他們自訂）」

  [ 接受委託 ] [ 回覆報價 ] [ 拒絕 ]
```

### 附屬國日常

```
商人（Local 業務）：看到委託 → 估價 → 回覆
信使（Local 助理）：接受後 → 逐項預訂
  → 飯店 ✅ 已確認
  → 餐廳 ✅ 已確認
  → 遊覽車 ⏳ 等待回覆
  → 和服體驗 ✅ 已確認

冒險公會（我們）即時看到所有狀態更新 🎉
```

### 附屬國金幣流

```
冒險公會 ──委託──→ 附屬國 ──預訂──→ 工匠（飯店/餐廳）
冒險公會 ←─報價─── 附屬國
冒險公會 ──付金幣→ 附屬國 ──付金幣→ 工匠
                   附屬國 ──請款──→ 冒險公會
                   附屬國 ←─收款─── 冒險公會
```
