## 📋 流程暗號

| 暗號         | 意思                              |
| ------------ | --------------------------------- |
| **開團**     | 勇者建立新的 tour                 |
| **種樹**     | 把行程寫進核心表                  |
| **發委託**   | 勾選項目 → 發需求單給供應商       |
| **打包委託** | 多項目打包發給同一個 Local        |
| **回填**     | 供應商回覆價格 → 修正世界樹       |
| **團確**     | 所有 confirmed → 給領隊的作戰地圖 |
| **落葉結算** | 回團後 → 實際 vs 估價 → 賺賠      |
