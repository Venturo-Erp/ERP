## 🔗 跨公司協作範例

### 範例 1：新功能開發

1. **William 🔱** 提出需求：「我們需要供應商評價系統」
2. **Caesar 🏛️** 規劃功能：User Stories、Wireframe
3. **UI-設計師 🎨** 設計介面：評價表單、列表頁
4. **前端工程師 🖥️** 開發功能：React 元件、API 串接
5. **IT-代碼審查員 👁️** Code Review：品質檢查
6. **Matthew 🔧** 部署上線：Vercel 部署
7. **Leon 📋** 開始使用：供應商評價

### 範例 2：行銷活動

1. **William 🔱** 決定：「推廣 ERP 給中小旅行社」
2. **Ben 🤝** 規劃業務策略：目標客群、話術
3. **IG-內容策展 📸** 建立內容：案例分享、功能介紹
4. **廣告投放經理 📱** 投放廣告：Meta Ads、目標受眾
5. **短視頻製作人 🎵** 製作短視頻：快速功能展示
6. **Ben 🤝** 接觸客戶：電話、Email 跟進

### 範例 3：系統安全升級

1. **IT-安全工程師 🔒** 發現漏洞：npm 套件過期
2. **Matthew 🔧** 評估影響：哪些功能受影響
3. **IT-自動化工程師 ⚙️** 建立自動化：定期掃描
4. **前端工程師 🖥️** 更新套件：升級依賴
5. **IT-代碼審查員 👁️** 驗證修復：確認無破壞性改動
6. **Matthew 🔧** 部署上線：灰度發布
