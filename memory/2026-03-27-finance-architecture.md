# 2026-03-27 財會架構討論

## 問題發現

### 1. 財務 vs 會計分層不清
- 財務報表依賴會計科目 → 沒設科目就看不到公司花了多少錢
- 應該分開：財務報表（依類別）+ 會計報表（依科目）

### 2. 缺少的功能
- 公司費用報表（不依賴會計科目）
- 預設項目種子資料（新租戶自動有預設設定）
- 報表項目篩選
- 年度比較

### 3. 系統性品質問題
- 今天發現的 SWR 閃爍問題 → 已修復 entity layer
- 應該有 Checklist 防止類似問題

## 正確架構

```
財務層（所有公司都用）
├─ 收支記錄（請款、收款）
├─ 現金流管理
├─ 應收/應付帳款
├─ 銀行對帳
└─ 財務報表（按類別分組，不需要會計科目）

會計層（需要的公司才開）
├─ 會計科目設定
├─ 傳票/分錄
└─ 會計報表（資產負債表、損益表）
```

## 預設項目（種子資料）

新租戶開戶時自動建立到「財務設定」：

| 類型 | 預設項目 |
|------|---------|
| 收款方式 | 現金、轉帳、信用卡、支票 |
| 付款方式 | 現金、轉帳、信用卡 |
| 團體請款類別 | 導遊費、餐費、門票、住宿、交通、保險 |
| 公司請款類別 | 出差費、交通費、行政雜支、廣告費 |
