## 🔄 完整生命週期

```
開團 → 排行程 → 報價 → 接單 → 需求（視情況）→ 確認 → 出團 → 回團 → 結帳 → 獎金
```

## 6️⃣ 確認（團確單）

**動作**：出團前確認所有細節

- 供應商回覆

- 預計成本

**產出**：

- 團確單 PDF

---

## 7️⃣ 出團 / 回團

**動作**：狀態變更

- `status: departed` → `returned`

---

## 8️⃣ 結帳

**動作**：結算團損益

- 收款（客戶）→ `receipts`
- 請款（供應商）→ `payment_requests` → `disbursement_orders`
- 計算損益

**流程**：

```
收款：
receipts (status=0 待確認) → 會計確認 → (status=1 已確認) → 自動過帳

請款：
payment_requests → 彙整成 disbursement_orders → 會計確認 → 付款
```

**問題**（已修復）：

- [x] 收款確認時 trigger 型別錯誤

**待確認**：

- [ ] 請款確認後的狀態變化？
- [ ] 團成本 vs 訂單成本 分開顯示？
