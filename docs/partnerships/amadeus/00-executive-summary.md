# Amadeus × Venturo 合作提案

## 執行摘要（一頁版）

**提案日期**：2026-04-18
**提案方**：Venturo（旅遊業 SaaS ERP 平台）
**接洽方**：Amadeus Taiwan
**核心訴求**：**一次性完整 Scope 合約**（避免日後分階段加價）

---

### 我們是誰

- **Venturo**：面向旅遊業的 SaaS ERP 平台，主打「旅遊業最強的後盾」
- **母公司 Corner**：營運 5 年的中高端客製化旅行社 + MICE 服務
- 創辦人 William 旅遊業資歷 12 年

### 我們要解決的問題

台灣 **3,000+ 家中小型旅行社**長期面對的三重痛點：

1. **無 IATA 資格** → 查位子、訂票要靠大票代
2. **不會操作 GDS 指令** → 業務訓練成本高
3. **訂票、ERP、對帳分散** → 資料在 4-5 個系統間切換

### 我們的解決方案

**Venturo ERP 內建 Amadeus GDS 模組**，提供雙模式介面：

- **簡單模式**：像 C 端消費者網站，新手業務立即上手
- **專業模式**：GDS 指令 / 快捷鍵，資深業務加速作業

訂單、異動、出票全部自動流入 Venturo ERP，解決資料分散。

### 殺手功能：航班異動自動通知

整合 Amadeus Queue / Schedule Recovery，小旅行社不再需要 24 小時人工巡 Queue。

### 向 Amadeus 的核心訴求

1. **一次性完整 API 授權合約**（避免日後逐項加價）
2. **多租戶 / SaaS 架構明確允許**（Venturo 授權下游小旅行社使用）
3. **合理的在地費率**（針對台灣中小旅行社市場）
4. **技術支援窗口 + Sandbox 環境**

### 給 Amadeus 的價值

- **打入 Amadeus 原本難以直接服務的長尾市場**
- **繁中在地化由 Venturo 負擔**，降低 Amadeus 在地成本
- 第一年目標引流 **30-50 家小旅行社**上線，第三年目標 **300+ 家**
