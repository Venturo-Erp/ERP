### 命名慣例決策
ERP 既有 trigger 主流命名是 `update_{table}_updated_at`（attractions / companies / customers / orders / tours 等）。所以兩張表各保留 `update_xxx_updated_at` 那條：

| Table | 保留 | 砍掉 |
|---|---|---|
| payment_requests | `update_payment_requests_updated_at` | `trigger_payment_requests_updated_at` |
| suppliers | `update_suppliers_updated_at` | `suppliers_updated_at_trigger` |
