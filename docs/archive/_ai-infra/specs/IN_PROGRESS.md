## 如何新增 Feature Spec

1. 跟 William 討論需求
2. 建立 `ai/specs/feature-name.md`
3. 更新這個 `IN_PROGRESS.md`
4. 準備開發時 → 精簡成 `ai/tasks/CURRENT_TASK.md`
