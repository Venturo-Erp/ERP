## 📊 完整流程檢查

### 階段 1：建立行程
```
TourItineraryTab
  ↓ 新增景點/餐食/住宿/交通
  ↓
tour_itinerary_items（建立 row）
  ├─ title, description, category, sub_category
  ├─ service_date, day_number
  ├─ resource_type, resource_id（連結 hotel/restaurant）
  └─ unit_price = null（尚無報價）

**檢查點**：
- ✅ 建立時有寫入核心表
- ✅ resource_id 有連結到對應的資源表（hotels, restaurants）

### 階段 3：發送需求單
```

  ↓

  ├─ tour_id, request_type, supplier_name

**檢查點**：
- ✅ 需求單有連結核心表（source_id）
- ✅ 核心表有反向連結（request_id）
